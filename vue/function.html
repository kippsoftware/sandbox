UX Functions

Math. A function is really a set. f(x) = y implies {x,y} is in the set f.

C. A function is really a procedure. You call f with parameter x. Statements within f execute. f returns y. Procedures can have side effects.

cfront. A method is a function. o.f(x) = y is semantically identical to f(o,x) = y.

Typed. Function has an aggregate type. Takes types t1, ... tn-1; returns tn

Interpreted. A function is a variable. f(x) = y. f evaluated with x returns y.

Script. A function is an object. f(x) = y implies f.invoke(x) returns y.

Object. Pass around functions like any other variable. 


function f(x) {
  return x + 1;
}

f = function(x) {
  return x + 1;
}

f = (x) => x + 1;
